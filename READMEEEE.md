# 🌱 Agri ML Service

A Spring Boot application that uses lightweight machine learning logic to:

- 🌾 Predict crop yield  
- 🍃 Detect plant diseases  

This is a starter ML-integrated backend project. In production, real models (ONNX, TensorFlow, or external ML APIs) can be integrated.

---

## ⚙️ Tech Stack

- Java 17  
- Spring Boot  
- Maven  
- REST APIs  
- Optional: Python (for dataset)

---

## 🚀 Build & Run

```bash
mvn clean package
mvn spring-boot:run
