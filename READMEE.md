# 🌱 Agri ML Service

Agri ML Service is a Spring Boot application that uses small embedded machine learning models to:

- Predict **crop yields** based on agro-climatic and management features  
- Detect **plant diseases** using environmental conditions and symptom inputs  

This project is designed for learning and demonstration purposes. In real-world scenarios, these models can be replaced with trained models (ONNX, TensorFlow, or external ML APIs).

---

## ✨ Features

- Crop yield prediction  
- Plant disease detection  
- Simple ML logic embedded in backend  
- REST API-based architecture  
- CRUD operations for crop records  

---

## 🛠️ Tech Stack

- Java 17  
- Spring Boot  
- Spring Web  
- Maven  
- MySQL (optional for persistence)  

---
